# Parts List

## Required Components

1. **Arduino Uno R3** (1x)
   - Microcontroller board

2. **16x2 LCD Display** (1x)
   - HD44780 controller
   - White text on blue backlight
   - 16 pins

3. **10kΩ Potentiometer** (2x)
   - Pot 1: LCD contrast adjustment (hardware)
   - Pot 2: Scroll speed control (read by Arduino on A1)

4. **220Ω Resistor** (4x)
   - 1 for LCD backlight current limiting
   - 3 for LED current limiting (one per LED)

5. **LED** (3x)
   - Suggested colours: red, yellow, green
   - Standard 5mm through-hole

6. **Breadboard** (1x)
   - Full-size recommended (more room for pots and LEDs)

7. **Jumper Wires** (approx. 20-25)
   - Male-to-male for breadboard connections
   - Various lengths recommended

## Optional Components

- **USB Cable (Type A to Type B)** - for programming and power
- **9V Battery + Battery Connector** - for portable power (optional)
- **Case/Enclosure** - to protect the project

## Tools Needed

- Arduino IDE (software - free download from arduino.cc)
- Computer with USB port
- Wire strippers (if using solid core wire)

## Estimated Cost

~$30-40 USD for all components if purchased separately
(Prices vary by supplier and region)

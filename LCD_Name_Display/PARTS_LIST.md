# Parts List

## Required Components

1. **Arduino Uno R3** (1x)
   - Microcontroller board

2. **16x2 LCD Display** (1x)
   - HD44780 controller
   - White text on blue backlight
   - 16 pins

3. **10kΩ Potentiometer** (1x)
   - For LCD contrast adjustment

4. **220Ω Resistor** (1x)
   - For LCD backlight current limiting

5. **Breadboard** (1x)
   - Half-size or full-size

6. **Jumper Wires** (approx. 16-20)
   - Male-to-male for breadboard connections
   - Various lengths recommended

## Optional Components

- **USB Cable (Type A to Type B)** - for programming and power
- **9V Battery + Battery Connector** - for portable power (optional)
- **Case/Enclosure** - to protect the project

## Tools Needed

- Arduino IDE (software - free download from arduino.cc)
- Computer with USB port
- Wire strippers (if using solid core wire)

## Estimated Cost

~$25-35 USD for all components if purchased separately
(Prices vary by supplier and region)
